I am building a CRM system in phases. Currently trying to enhance the referral relationship management. The Requirements Document: Referral Tree & Genealogy Page (Feng Shui CRM) – Vertical Layout
This document defines the complete requirements for the Referral Tree & Genealogy Page within the Referral Relationship Management module of the Feng Shui CRM. The page is designed with a vertical layout (summary and leaderboard above the tree) to ensure usability on mobile devices, while remaining clear on desktop.

IMPORTANT RULES:
1. BUILD UPON my existing code - do NOT recreate previous phases
2. PRESERVE ALL existing functionality - all previous phases must still work
3. ONLY ADD new code for this phase

1. Business Context
The CRM supports a Feng Shui consultancy. Key entities:

Prospects: Potential clients created via CPS activities or manual entry.

Customers: Converted from prospects after a purchase (or manually).

Referrals: Links between a referrer (customer) and a referred prospect.
Note: Only customers can be referrers in the current schema. However, the requirement that “referral is for PROSPECT only” implies prospects may also refer. For this version, we will extend the data model to allow prospects as referrers (adding referrer_prospect_id and referrer_type). The tree and summary will treat both customers and prospects as referrers.

The referral tree shows the downline of any selected person (prospect or customer). The root can be the logged‑in user (for lower roles) or any member the user is authorised to view (for higher roles).

2. User Roles & Access
All 13 roles have access to the Referral Relationship Management tab, but visibility is controlled:

5. Referral Performance Summary (Last 6 Months)
The summary section appears above the tree and includes:

5.1 Performance Metrics
Metric	Calculation
Total Prospects Who Referred	Count of distinct prospects (or customers) who have referred at least one person in the last 6 months (based on referrals created in that period).
Total Referrals Made	Count of all referrals created in the last 6 months.
Referrals Converted to Customers	Count of referrals where the referred_prospect_id has a corresponding customer record, and either the conversion date is within the last 6 months or the referral was made within the last 6 months and the prospect has been converted at any time.
Conversion Rate	(Referrals Converted / Total Referrals) * 100 (rounded to one decimal).

9.2 Referral Tree Building
A function buildReferralTree(rootId, rootType):

Retrieves the person from the appropriate table.

Queries referrals where referrer_id = rootId and referrer_type = rootType.

For each referral, fetches the referred prospect (from prospects), then recursively builds children (if that referred prospect has become a customer, it may be the referrer for others; we need to treat customers as referrers as well). The recursion continues.

9.3 Colour Calculation Helpers
getProspectColour(prospect):

Check referrals where this prospect is the referrer (referrer_id = prospect.id, referrer_type = 'prospect') and the referral date is within 3 months → green.

Else, get latest activity date from activities → assign blue/orange/yellow/gray.

getCustomerBadge(customer):

Get latest purchase date from purchases → assign green/blue/orange/purple/gray badge.

9.4 Summary Calculation
Query referrals where created_at >= (today - 6 months).

Count distinct referrers (using referrer_id and referrer_type) → “Total Prospects Who Referred” (this includes both prospects and customers, but we count them as “referrers”).

Count total referrals.

For each referral, check if the referred prospect has a customer record → count conversions.

Compute conversion rate.

9.5 Top Referrers
Aggregate referrals by referrer_id, referrer_type for the last 6 months.

Sort by count descending, take first 3.

For each, retrieve the name from the respective table (prospects or customers).

// Ensure referrals have the new fields (id, referrer_id, referrer_type, referred_prospect_id)
const ensureReferralFields = () => {
    const referrals = DataStore.getAll('referrals');
    let changed = false;
    referrals.forEach(r => {
        if (r.referrer_customer_id && !r.referrer_id) {
            // Old format: convert
            r.referrer_id = r.referrer_customer_id;
            r.referrer_type = 'customer';
            delete r.referrer_customer_id; // optional, but we keep for backward compat
            changed = true;
        }
        if (!r.referrer_id) {
            // Should not happen, but just in case
            r.referrer_id = null;
            r.referrer_type = null;
        }
        if (!r.created_at) r.created_at = r.date || new Date().toISOString();
    });
    if (changed) {
        // Persist changes
        safeLocalStorage.set('fs_crm_referrals', referrals);
    }
};

const renderReferralSummaryAndLeaderboard = () => {
    renderSummary();

const renderSummary = () => {
    const summaryDiv = document.getElementById('referral-summary');
    if (!summaryDiv) return;
    const sixMonthsAgo = new Date();
    sixMonthsAgo.setMonth(sixMonthsAgo.getMonth() - 6);
    const referrals = DataStore.getAll('referrals').filter(r => new Date(r.created_at) >= sixMonthsAgo);

    // Distinct referrers
    const referrersMap = new Map();
    referrals.forEach(r => {
        const key = `${r.referrer_type}:${r.referrer_id}`;
        if (!referrersMap.has(key)) {
            referrersMap.set(key, { type: r.referrer_type, id: r.referrer_id, count: 0 });
        }
        referrersMap.get(key).count++;
    });
    const totalReferrers = referrersMap.size;

    // Total referrals
    const totalReferrals = referrals.length;

    // Converted referrals
    const converted = referrals.filter(r => {
        const prospect = DataStore.getById('prospects', r.referred_prospect_id);
        return prospect && prospect.status === 'converted';
    }).length;

    const conversionRate = totalReferrals ? ((converted / totalReferrals) * 100).toFixed(1) : 0;

    // Top 3 referrers (by count)
    const topReferrers = Array.from(referrersMap.values())
        .sort((a, b) => b.count - a.count)
        .slice(0, 3)
        .map(entry => {
            let name = '';
            if (entry.type === 'customer') {
                const c = DataStore.getById('customers', entry.id);
                name = c ? c.full_name : `Customer ${entry.id}`;
            } else {
                const p = DataStore.getById('prospects', entry.id);
                name = p ? p.full_name : `Prospect ${entry.id}`;
            }
            return { ...entry, name };
        });

    summaryDiv.innerHTML = `
        <div class="summary-cards">
            <div class="summary-card">
                <div class="card-title">Total Prospects Who Referred</div>
                <div class="card-value">${totalReferrers}</div>
            </div>
            <div class="summary-card">
                <div class="card-title">Total Referrals Made</div>
                <div class="card-value">${totalReferrals}</div>
            </div>
            <div class="summary-card">
                <div class="card-title">Referrals Converted to Customers</div>
                <div class="card-value">${converted}</div>
            </div>
            <div class="summary-card">
                <div class="card-title">Conversion Rate</div>
                <div class="card-value">${conversionRate}%</div>
            </div>
        </div>
        <div class="top-referrers">
            <h4>🏆 Top Referrers (Last 6 Months)</h4>
            <div class="top-referrers-list">
                ${topReferrers.map((t, idx) => `
                    <div class="top-referrer-item" data-id="${t.id}" data-type="${t.type}">
                        <span class="rank">${idx+1}</span>
                        <span class="name">${escapeHtml(t.name)}</span>
                        <span class="count">${t.count} referral${t.count !== 1 ? 's' : ''}</span>
                    </div>
                `).join('')}
                ${topReferrers.length === 0 ? '<div class="no-data">No referrals in last 6 months</div>' : ''}
            </div>
        </div>
    `;

    // Attach click handlers for top referrers
    document.querySelectorAll('.top-referrer-item').forEach(el => {
        el.onclick = () => {
            const id = parseInt(el.dataset.id);
            const type = el.dataset.type;
            showReferralTree(id, type);
        };
    });
};

const renderLeaderboard = () => {
    const container = document.getElementById('leaderboard-list');
    if (!container) return;

    const period = document.getElementById('leaderboard-period')?.value || '6months';
    const showHidden = document.getElementById('show-hidden-toggle')?.checked || false;
    const userId = _currentUser?.id || 'guest';
    const hiddenKey = `hidden_referrers_${userId}`;
    let hidden = safeLocalStorage.getJSON(hiddenKey, []);

    // Filter referrals by period
    const now = new Date();
    let cutoff = null;
    if (period === 'year') cutoff = new Date(now.getFullYear(), 0, 1);
    else if (period === '6months') { cutoff = new Date(); cutoff.setMonth(now.getMonth() - 6); }
    else if (period === 'month') cutoff = new Date(now.getFullYear(), now.getMonth(), 1);

    let referrals = DataStore.getAll('referrals');
    if (cutoff) referrals = referrals.filter(r => new Date(r.created_at) >= cutoff);

    // Group by referrer
    const referrerMap = new Map();
    referrals.forEach(r => {
        const key = `${r.referrer_type}:${r.referrer_id}`;
        if (!referrerMap.has(key)) {
            referrerMap.set(key, { type: r.referrer_type, id: r.referrer_id, count: 0, lastDate: r.created_at });
        }
        const entry = referrerMap.get(key);
        entry.count++;
        if (new Date(r.created_at) > new Date(entry.lastDate)) entry.lastDate = r.created_at;
    });

    const sorted = Array.from(referrerMap.values()).sort((a, b) => b.count - a.count);
    const top10 = sorted.slice(0, 10);

    let html = '';
    top10.forEach((item, idx) => {
        const isHidden = hidden.some(h => h.id === item.id && h.type === item.type);
        if (isHidden && !showHidden) return;

        let name = '';
        if (item.type === 'customer') {
            const c = DataStore.getById('customers', item.id);
            name = c ? c.full_name : `Customer ${item.id}`;
        } else {
            const p = DataStore.getById('prospects', item.id);
            name = p ? p.full_name : `Prospect ${item.id}`;
        }

        html += `
            <div class="leaderboard-item rank-${idx+1} ${isHidden ? 'hidden-item' : ''}">
                <div class="rank-badge">${idx+1}</div>
                <div class="referrer-info" data-id="${item.id}" data-type="${item.type}">
                    <div class="referrer-name">${escapeHtml(name)}</div>
                    <div class="referrer-stats">${item.count} referrals | Last: ${item.lastDate.substring(0,10)}</div>
                </div>
                <div class="hide-btn" data-id="${item.id}" data-type="${item.type}">
                    <i class="far ${isHidden ? 'fa-eye' : 'fa-eye-slash'}"></i>
                </div>
            </div>
        `;
    });

    if (html === '') html = '<div class="no-data">No referrers found</div>';
    container.innerHTML = html;

    // Attach click handlers for referrer info (to show tree)
    container.querySelectorAll('.referrer-info').forEach(el => {
        el.onclick = () => {
            const id = parseInt(el.dataset.id);
            const type = el.dataset.type;
            showReferralTree(id, type);
        };
    });
    // Attach hide/unhide handlers
    container.querySelectorAll('.hide-btn').forEach(btn => {
        btn.onclick = (e) => {
            e.stopPropagation();
            const id = parseInt(btn.dataset.id);
            const type = btn.dataset.type;
            toggleHideReferrer(id, type);
        };
    });
};

const toggleHideReferrer = (id, type) => {
    const userId = _currentUser?.id || 'guest';
    const hiddenKey = `hidden_referrers_${userId}`;
    let hidden = safeLocalStorage.getJSON(hiddenKey, []);
    const existingIndex = hidden.findIndex(h => h.id === id && h.type === type);
    if (existingIndex !== -1) {
        hidden.splice(existingIndex, 1);
    } else {
        hidden.push({ id, type });
    }
    safeLocalStorage.set(hiddenKey, hidden);
    renderLeaderboard();
    UI.toast.info('Hidden preferences updated.');
};

const buildTreeData = (rootId, rootType) => {
    // Root person data
    let rootPerson = null;
    if (rootType === 'customer') rootPerson = DataStore.getById('customers', rootId);
    else rootPerson = DataStore.getById('prospects', rootId);
    if (!rootPerson) return null;

    const node = {
        id: rootPerson.id,
        name: rootPerson.full_name,
        type: rootType,
        // Additional metadata for rendering
        colour: null,
        badge: null,
        isRecentReferrer: false,
        latestNote: null
    };

    // For prospect: determine colour
    if (rootType === 'prospect') {
        node.colour = getProspectColour(rootPerson);
    } else {
        node.badge = getCustomerBadge(rootPerson);
    }

    // Get referrals made by this person
    const referrals = DataStore.getAll('referrals').filter(r =>
        r.referrer_id === rootId && r.referrer_type === rootType
    );
    node.children = [];
    for (const ref of referrals) {
        const childProspect = DataStore.getById('prospects', ref.referred_prospect_id);
        if (!childProspect) continue;
        // Recursively build child (if the referred prospect has become a customer, it can be a referrer)
        let childType = 'prospect';
        const childCustomer = DataStore.getAll('customers').find(c => c.converted_from_prospect_id === childProspect.id);
        if (childCustomer) childType = 'customer';
        const childNode = buildTreeData(childProspect.id, childType);
        if (childNode) {
            // Add referral details
            childNode.referralSource = ref.referral_source;
            childNode.referralDate = ref.created_at;
            node.children.push(childNode);
        }
    }

    // Also, if root is a customer, we might want to show that they were referred by someone (parent). Not needed for tree.

    return node;
};

const getProspectColour = (prospect) => {
    // Check if prospect referred someone in last 3 months
    const threeMonthsAgo = new Date();
    threeMonthsAgo.setMonth(threeMonthsAgo.getMonth() - 3);
    const referrals = DataStore.getAll('referrals').filter(r =>
        r.referrer_id === prospect.id && r.referrer_type === 'prospect' &&
        new Date(r.created_at) >= threeMonthsAgo
    );
    if (referrals.length > 0) return 'green'; // 🔥

    // Else get last activity
    const activities = DataStore.getAll('activities').filter(a => a.prospect_id === prospect.id);
    if (activities.length === 0) return 'gray';
    const lastActivity = activities.sort((a,b) => new Date(b.activity_date) - new Date(a.activity_date))[0];
    const daysSince = Math.floor((Date.now() - new Date(lastActivity.activity_date)) / (1000*60*60*24));
    if (daysSince <= 30) return 'blue';
    if (daysSince <= 90) return 'orange';
    if (daysSince <= 180) return 'yellow';
    return 'gray';
};

const getCustomerBadge = (customer) => {
    const purchases = DataStore.getAll('purchases').filter(p => p.customer_id === customer.id);
    if (purchases.length === 0) return 'gray';
    const lastPurchase = purchases.sort((a,b) => new Date(b.date) - new Date(a.date))[0];
    const daysSince = Math.floor((Date.now() - new Date(lastPurchase.date)) / (1000*60*60*24));
    if (daysSince <= 90) return 'green';
    if (daysSince <= 180) return 'blue';
    if (daysSince <= 365) return 'orange';
    if (daysSince <= 730) return 'purple';
    return 'gray';
};
