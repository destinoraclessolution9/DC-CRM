Core sales workflow

Step 01

Sign in & find your way around

Everyone

Open the CRM, sign in with the email your manager set up. You land on Calendar by default. Dark sidebar on the left switches modules — 12 in total. Top-right initials open profile / sign-out.

↓ Download GIF ⎘ Copy link
Step 02

Add a new prospect

Everyone

Prospects → + Add Prospect. Only Name and Phone are required. Pick the Source accurately — it powers campaign ROI numbers later. Save closes the modal and the new row appears at the top.

↓ Download GIF ⎘ Copy link
Step 03

Move a deal through the pipeline

Sales + managers

Kanban with five columns left→right: New → Contacted → Qualified → Proposal → Won. After every call, drag the card one column right. Click any card for the deal detail and the green ★ Mark Won button.

↓ Download GIF ⎘ Copy link
Step 04

Schedule a calendar booking

Everyone

Click any empty day cell — modal opens with the date pre-filled. Customer field auto-suggests existing prospects. Save creates a calendar entry AND links it to the customer's history. SMS reminder fires 24h before.

↓ Download GIF ⎘ Copy link
Step 05

Open a case and log follow-ups

Everyone

Cases tracks customer problems until they're closed. + New Case, write a specific Subject, set Priority + Owner. On the case page, every follow-up becomes a timestamped row. When fixed, hit green Mark Resolved.

↓ Download GIF ⎘ Copy link
Step 06

Add a referral & read the tree

Everyone

Log a referral the moment you hear about it. Switch to Tree view to see who-referred-whom — top referrers in pink, direct in purple, second-level in blue. Compounding referral networks are how we grow.

↓ Download GIF ⎘ Copy link
Step 07

Read performance & export a report

Everyone (filtered by role)

Performance is your scoreboard — KPIs vs target this month. Numbers under target turn orange; below 50% red. Reports → Run → Download (XLSX is the team default, includes chart preview).

↓ Download GIF ⎘ Copy link
Step 08

Upload to Knowledge HQ

Everyone

Team library — cold-call scripts, refund SOPs, qualification checklists. + Upload, drop file, fill Title + Tags. Inside any doc, Share copies a deep link for WhatsApp. Edits keep version history.

↓ Download GIF ⎘ Copy link

Campaigns & team

Step 09

Create a promo code

Marketing Manager / Super Admin

+ New Promo → short code (e.g. JUNE20), % or RM off, valid dates, optional usage cap. Share card gives three formats: Copy code, Download PNG flyer, WhatsApp.

Step 10

Build a marketing-automation rule

Marketing Manager / Super Admin

Three steps: When (trigger), If (condition), Then (action). Defaults to Paused so you can dry-run before going live. Open any rule for the run log — Delivered / Failed / Opt-outs.

Step 11

Manage the team — Agents & roles

Super Admin / Marketing Manager

List every team member with role badge + this month's deal count. + Add Agent invites a new joiner — they get an email to set their password. Roles: Super Admin / Marketing Manager / Manager / Agent.

Documents library

Customer-facing file store. Folders by type. Link to customer in the upload modal makes the file show in their profile too. Send via WhatsApp pushes the file to the linked customer's number.

AI suite

Step 13

AI Insights dashboard

Manager and above

Your morning brief. Six colour-coded cards highlight what the model spotted overnight — hot leads, churn risk, pipeline, agent patterns, best-call-time. Click Open on any card to drill in.

↓ Download GIF ⎘ Copy link
Step 14

Lead Scoring

Everyone

Every prospect gets a 0–100 AI score. Hot (80+) / Warm (50–79) / Cold (<50). Trend column shows if the score moved this week. Drill in to see what's pushing it up or down + Next-best-action button.

↓ Download GIF ⎘ Copy link
Step 15

Sales Forecast

Manager and above

AI-projected revenue 90 days out. Locked / Likely / At-risk / Stretch buckets. Chart with most-likely line + optimistic + pessimistic confidence band. ~84% accurate on 60-day calls.

↓ Download GIF ⎘ Copy link
Step 16

Churn Risk Analysis

Manager and above

Customers about to leave. Risk score with reasons (no contact 90d, sentiment drop, late payment, etc.). Save action plan ticks one or more rescue actions and creates Cases with you as owner.

Step 17

Manager / Marketing Manager only

Model watches your team and surfaces patterns worth copying or fixing. Each tip has a finding, a why, and a one-click Apply action (extract template, schedule 1-on-1, add KPI tile). Reversible.

Security & operations

Step 27

Set up two-factor auth

Everyone (strongly recommended)

Four steps: pick method → scan QR with Authy / Google Authenticator / 1Password → type the 6-digit code → save 10 one-time backup codes. Single most useful security step — takes 2 minutes.

Mobile & performance

Step 31

Mobile calendar — cold paint

Everyone on mobile · helpful for devs & managers debugging

What the calendar looks like when you tap it on your phone (cold open). Tier 1.1 splits the load into Phase 1 (activities — ~400 ms, grid is usable straight away) and Phase 2 (names, birthdays, the Coming up strip — silently fills in ~1.2 s later). First-interactive is ~5.8× faster than before. If Phase 1 consistently takes more than a second on your phone, ping the dev team — likely a network issue, not the CRM.
