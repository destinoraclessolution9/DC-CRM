## ISSUE TYPE: Bug Report / Feature Request

### MODULE: Calendar Module (Phase 1)

### CURRENT BEHAVIOR:
1. Calendar is hardcoded to only show March 2026
2. Filter button does not open any filter modal
3. Quick Add Activity button is placed below the calendar instead of beside the filter button
4. Calendar shows Sunday as first day (needs Monday first)

### EXPECTED BEHAVIOR:
1. Dynamic Calendar Range
   - Calendar should work for ANY date from year 2010 to 2200
   - Users should be able to navigate to any month/year using Previous/Next buttons
   - "Today" button should jump to current date
   - Date calculation should be dynamic, not hardcoded

2. Calendar Display Format
   - Week should start on Monday (not Sunday)
   - Days should be labeled: Mon, Tue, Wed, Thu, Fri, Sat, Sun
   - All months should render correctly with proper number of days
   - Leap years should be handled correctly (Feb 29)

3. Filter Button Functionality
   - Clicking Filter should open filter modal
   - Filter options should include:
     * Filter by Agent (dropdown)
     * Filter by Activity Type (CPS, FTF, FSA, EVENT, etc.)
     * Date Range selector (from date - to date)
     * Search by activity title
   - Filters should apply to both calendar view and Today's Activities
   - Filter state should persist during session

4. UI Layout
   - Quick Add Activity button should be moved to the right side of the filter button
   - Layout should be: [Calendar Title] [Today] [Day] [Week] [Month] [Filter] [Quick Add]
   - Buttons should be properly aligned in the same row

5. Today's Activities Section
   - Should show activities for the currently selected date
   - Should update when navigating to different dates
   - Activity cards should show correct time, type, and customer info

### TECHNICAL REQUIREMENTS:
Reference: Website Requirements Document V8.6 - Section 5: Calendar & Activity Management
- Calendar should support years 2010-2200
- Activities should be loaded from DataStore dynamically
- Filter state should be saved in sessionStorage
- Monday first day of week (ISO standard)

### VISUAL REFERENCE:
Current: Filter button not working, Quick Add below calendar
Expected: Filter and Quick Add buttons side by side in header

### PRIORITY: High
