# DNJ TikTok (Melayu) — Caption + Skrip Bank v2 (Hari 1–30)

**Framework:** [marketing-principles.md](marketing-principles.md) — every script ikut **Hook → Pain → Solution → CTA**.

Setiap hari ada:
- **HOOK (1.5s)** — apa yang stop scroll
- **PAIN (5–8s)** — cermin emosi audiens
- **SOLUTION (10–18s)** — tactical, actionable, ada nombor
- **CTA (3–5s)** — ONE specific action
- **On-screen text** (cover hook, mesti ada nombor)
- **Caption** (search-led, ringkas)
- **Hashtag** (komen pertama, 5 max)

> ⚠️ **Garisan merah:** Tiada ayat ini boleh disebut walaupun joke — bomoh, tilik, ramalan, numerologi, horoskop, zodiak, feng shui, BaZi, life chart, mistik. Sebut "AI matching" sahaja.

# Cheatsheet — guna setiap hari

## Pre-post checklist (7 tick mandatory)
- [ ] Topic = search-led (TikTok autocomplete check)
- [ ] Hook = stop scroll 1.5s (number / question / contrarian)
- [ ] Pain = specific scenario, audience cermin
- [ ] Solution = TACTICAL (steps + numbers), bukan motivasi
- [ ] CTA = ONE specific action (save / DM keyword / tag / komen)
- [ ] Cover = big number + bright accent
- [ ] BaZi check = zero ramalan/tilik/horoskop language

## CTA bank (rotate untuk variety)
- **Save authority:** "Save 📌 — guna esok bila kau [scenario]."
- **DM keyword:** "DM **'X'** — aku send [resource] direct."
- **Tag:** "Tag siapa yang [situation] — diorang patut tahu ni."
- **Komen:** "Komen NOMBOR — yang mana paling kena."
- **Follow Part 2:** "Follow — Part 2 esok dengan [specific value]."

## Hot keyword Malay yang validate weekly
`real / sebenar` · `self-check` · `step by step` · `data` · `insider` · `HR cakap` · `Save 📌` · `DM 'X'`
