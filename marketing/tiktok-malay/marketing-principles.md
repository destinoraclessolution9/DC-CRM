# DNJ TikTok (Melayu) — Marketing Principles v2

**Source:** 小红书疯狂获客实战营 — Chalette 女皇 (RedNote客 fanatic acquisition bootcamp).
**Adapted for:** Malay TikTok market, DNJ recruitment platform.
**Date:** 2026-05-29.

Why v2: My v1 plan was brand-led ("aku buat akaun ni sebab…"). Chalette's framework demands **search-led** content (what customers actually search for) + a strict **Hook→Pain→Solution→CTA** copy formula on every video. This file documents the upgrade so future iterations stay aligned.

---

## 1. The positioning formula

**Positioning = Industry × Monetization Model**

| For DNJ |  |
|--|--|
| **Industry** | Recruitment / career / job-search (大健康 of Malaysian work life) |
| **Monetization** | Virtual service — AI matching platform (free user-side; B2B revenue from employers) |
| **Differentiation** | Tak kekurangan platform kerja. Yang kurang = **kepercayaan + suara yang faham**. We sell trust, not job board features. |

**Implication:** Every video must build **trust + influence**, not push the product. The product CTA comes only after 3+ touches of value/empathy.

---

## 2. The 4-step content acquisition model

> Every TikTok must answer: 选题 (topic) → 文案 (copy) → 图文/视频 (visual) → 封面 (cover/thumbnail).

| Step | Question | Failure mode |
|--|--|--|
| ① **Topic** | Does the customer SEARCH for this? | Posting what *you* want to say (self-entertainment) |
| ② **Copy** | Does it follow Hook→Pain→Solution→CTA? | Pain dwell + no actionable solution + weak CTA |
| ③ **Visual/Video** | Is it native to TikTok Malay? | Looks like corporate ad, no subtitle, no trending sound |
| ④ **Cover** | Does it scream "click me" in 0.5s? | Small font, no number, no contrast |

### ▶ Topic selection — search-led, not brand-led

**Wrong (v1, brand-led):**
- "Aku buat akaun ni sebab 73% fresh grad…"
- "POV: kau hantar 100 resume"
- "Storytime: kerja pertama aku gaji RM1,800"

**Right (v2, search-led):**
- "Fresh grad Malaysia 2026 — berapa resume nak hantar sebelum dapat offer? Data sebenar."
- "100 resume 0 reply? 5 thing korang check dulu sebelum give up 🙏"
- "Gaji RM1,800 di KL — boleh survive? Real math + 3 langkah keluar."

**The recipe (3 steps to find search-led topics):**

1. Open TikTok → search bar → type **"cari kerja"**, **"resume"**, **"gaji"**, **"interview"** → screenshot the **autocomplete suggestions**. Those are what users actually search.
2. Search each one → sort by Most Liked → identify the top 3 viral formats in that vertical.
3. Reverse-engineer the structure (hook pattern, pain point, solution angle) → adapt to DNJ's voice.

**Templates that always work (adapt with new pain points):**

- "X thing dalam YYY yang ramai tak perasan 🤯"
- "Kalau korang kerja [vertical], tengok ni dulu"
- "Self-check: 5 tanda kau [pain state]"
- "Aku cek data Z graduan/HR/resume — ni pattern yang aku jumpa"
- "[Demographic] di Malaysia, save ni — guna time [scenario]"

### ▶ Copy — the 4-segment formula

Every script, every video, every length:

```
HOOK     →  1.5 saat. Stop scroll. Soalan / nombor ganjil / claim provokatif.
PAIN     →  5–8 saat. Cermin emosi audiens. "Yes, kau faham aku."
SOLUTION →  10–18 saat. TACTICAL. Steps/tips/framework. Bukan motivasi.
CTA      →  3–5 saat. Action spesifik: SAVE / DM "X" / TAG kawan / Komen "Y".
```

**Hook patterns (steal these):**

| Pattern | Malay example |
|--|--|
| **Big number** | "73% fresh grad Malaysia hantar 50+ resume sebelum reply pertama." |
| **Contrarian claim** | "Stop hantar 100 resume. Hantar 10 yang betul = 4× reply rate." |
| **Specific identity** | "Kalau kau gaji bawah RM3,500 di KL — tengok ni dulu." |
| **Direct question** | "Berapa lama purata kau tunggu dari interview ke offer?" |
| **Self-check tease** | "Self-check 30 saat: kau underpaid atau market memang naik?" |
| **Insider reveal** | "Aku tanya 12 HR Malaysia — ni 3 thing yang buat resume terus reject." |

**Solution must be TACTICAL.** "Percaya diri kau" is not a solution. "Buka resume kau sekarang, padam 3 perkara ni" IS a solution.

**CTA must drive ONE specific action.** Not "follow + save + share + tag" (decision fatigue). Pick ONE per video and own it:

- **Build saves** (authority/tips video) → "Save ni 📌. Guna esok bila kau apply."
- **Build comments** (debate/community) → "Komen nombor mana paling kena pada korang."
- **Build follows** (story/serial) → "Follow untuk Part 2 esok."
- **Build DMs** (lead gen) → "DM 'PADANAN' aku send link untuk start."
- **Build shares** (emotional) → "Tag kawan yang tengah cari kerja."

### ▶ Visual — TikTok-native, not corporate

- 9:16, subtitle wajib, trending Malay sound, ≤30s sweet spot.
- Talking head: tengok kamera, jangan baca dari kertas (audience detect 1 saat).
- POV/slideshow: 1 idea per slide, big font, hero number on slide 1.
- B-roll: gunakan IG content folder bila boleh (`marketing/instagram/content/`).
- End frame: pakai `content/end-frame.png` selalu. Brand consistency = recognition.

### ▶ Cover/thumbnail — 4 mandatory elements

Every static cover (Hari 1, slideshow Day 1 of each deck, profile cover) MUST have:

| Element | Why | DNJ rule |
|--|--|--|
| **1. Big font** | Readable di feed kecil | Headline ≥120pt on 1080×1920 |
| **2. Bright contrast** | Stand out di FYP dark mode | Hitam BG + emas accent + putih body |
| **3. 3:4 or 9:16** | TikTok native | 9:16 (1080×1920) selalu |
| **4. Big number** | Brain auto-stop on numbers | Hero number ≥240pt — "73%", "RM2,500", "5 RED FLAG", "100 → 3" |

**3 cover formulas (adapted to DNJ verticals):**

| Vertical | Formula | Example |
|--|--|--|
| **Virtual service** (DNJ produk) | Resonance + Method + Result | "Apply 73 kerja → 0 reply? AI matching cuma 3." |
| **Real story** (Permata) | Before vs After + numbers + time | "Kedai mamak → Software Engineer. 24 bulan." |
| **Tips** (authority) | "X thing untuk YYY" + Save hint | "5 RED FLAG iklan kerja MY · SAVE 📌" |

## 3. The 3 drainage methods (导流)

Chalette's principle: every viral note must funnel attention into **owned channels** (private group, DM, website). Don't let traffic die in the comments.

**For DNJ TikTok Malay — adapted:**

### Method 1: Bio drainage (formula bio)

**Old bio (v1):**
> Bukan kau tak ada bakat 💎
> AI kita cari 3 kerja paling padan — bukan 100 reject.
> 🇲🇾 Malaysia · Pencari Kerja & Majikan
> 👇 diamondandjeweler.com

**Problem:** Pretty, but no clear funnel. No social proof. No specific action.

**New bio (v2 — Chalette's "who I am + who I help + how + result + CTA"):**
```
Hi, aku DNJ 💎 Cari kerja padan AI 🇲🇾
Bantu pencari kerja Malaysia → 3 padanan betul (bukan 100 reject)
1,000+ user dah dapat offer dalam 21 hari purata
👇 DM "PADANAN" / link bio untuk start free
```

Each line does one job:
1. Identity + niche keyword (search-found)
2. Who you help + specific result
3. Social proof (real number)
4. Specific CTA action

### Method 2: Comment / DM funnel (sebab fan group tak applicable di Malaysia)

TikTok Malaysia tak ada culture "fan group" macam 微信群 di China. Equivalent kita = **DM keyword funnel**.

Setiap minggu 1 video kena ada "DM keyword" CTA:
- "DM **'PADANAN'** — aku send link onboarding."
- "DM **'RESUME'** — aku review free dalam 48 jam."
- "DM **'GAJI'** — aku bagi salary benchmark Malaysia industry kau."

Auto-reply via TikTok DM template → push ke link / WhatsApp / website.

### Method 3: TikTok Ads on winning videos

Setelah satu video break 50K organic views, boost dengan **RM10–30/hari × 3 hari** dalam TikTok Ads Manager. Cuma untuk video yang dah viral organic — bukan untuk save bad video. (Cost: ~RM100 per 100K views, sangat reasonable di MY market.)

## 4. Income formula (Chalette's universal math)

```
Revenue = Traffic × Conversion × Average Order Value
```

**For DNJ (talent side — free, monetization on employer side):**
```
Employer leads = TikTok talent followers × Talent→Profile completion rate × Profile match rate × Employer subscription value
                 (engineered indirectly through talent acquisition)
```

**Direct metric to optimize on TikTok:**
- Followers → website signup → profile completion → matched candidate in DNJ pool.
- Target: 100 followers/day × 5% signup × 60% complete profile = **3 quality candidates/day**.

## 6. Quality control checklist — before posting

Tick all 7 sebelum hit publish:

- [ ] **Topic:** is this what users search? (Check TikTok search autocomplete)
- [ ] **Hook:** stop scroll dalam 1.5 saat? (Number / question / contrarian claim?)
- [ ] **Pain:** audience cermin emosi? (Specific scenario, not vague)
- [ ] **Solution:** TACTICAL — boleh actionable hari ni? (Bukan motivasi)
- [ ] **CTA:** ONE specific action? (Save / Comment X / DM Y / Tag)
- [ ] **Cover:** big number + bright contrast + readable di 0.5s?
- [ ] **BaZi check:** zero ramalan / tilik / horoskop language?

## 7. Reverse-engineering competitor virals (weekly habit)

**Setiap Ahad, 30 minit:**
1. Open TikTok search → "cari kerja Malaysia" / "fresh grad" / "interview tips"
2. Sort: Most Liked, Past 7 days
3. Screenshot top 5 videos: hook / structure / pain / solution / CTA
4. Identify the pattern repeated across 3+ videos
5. Adapt 1 video next week guna pattern tu — credit nobody, structure boleh dipinjam

**Tools (semua free):**
- TikTok native search + autocomplete
- TikTok Creative Center → Trends → Songs (untuk trending Malay sound)
- Google Trends Malaysia → category Career & Education → trending search
