## FASA 3 · Edit + Post (9 minit)

### 3.1 Buka CapCut / TikTok native editor

**Aset yang kena import:**
1. `marketing/tiktok-malay/content/hari1-thumbnail.png` — drop sebagai opening 2s B-roll punch (hero number "72")
2. Footage muka kau (talking-head 25s)
3. `marketing/tiktok-malay/content/end-frame.png` — drop sebagai last 3s

**Auto-caption (mandatory):**
- CapCut → Text → Auto Captions → Bahasa Malaysia
- Betulkan ejaan (TikTok auto-caption tak 100% accurate)
- Position: tengah, white text + black outline

**Trending sound:**
- TikTok library → search trending Malay sound (sad/inspirational mood)
- Volume: 15-20% (your voice dominant)

**Export:** 1080×1920, MP4, 30fps

### 3.2 Post di TikTok app

**Caption (paste ni):**
```
Real data, real graduan KL. 72 resume purata. 3 thing untuk pecah median. DM "DATA" untuk riset penuh 📊
```

**Pertama comment (paste hashtag ni dalam komen pertama, BUKAN caption):**
```
#FreshGradMalaysia #CariKerjaMalaysia #MalaysiaJobs #DNJmy #FYP
```

**Cover thumbnail:** Pilih yang ada teks "72" — Tap "Select cover" → drag to first frame yang tunjuk thumbnail
**Posting time:** 7:30–8:30pm MYT
**Settings:**
- Allow comments: ON
- Allow Stitch: ON (penting untuk reach + collab future)
- Allow Duet: ON
- Allow downloads: OFF (protect content)

### 3.3 Selepas post — FIRST HOUR ENGAGEMENT

**Critical untuk algorithm:**
- **Reply setiap komen dalam 5 minit pertama** — paling tinggi rank signal di TikTok
- Pin best comment ("save ni" / "follow") sendiri sebagai first comment
- Share post ke 5 chat WhatsApp peribadi (kawan-kawan first audience)
